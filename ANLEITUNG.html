🍦 Gmail Export Tool

Schritt-für-Schritt Anleitung für EisFavorite

1 Python & Pakete installieren

  • 📥 Falls noch nicht installiert: Python herunterladen
  • ⚠️ Bei Installation "Add Python to PATH" anhaken!
  • 🔧 Doppelklick auf INSTALL.bat
✅ Fertig? Du solltest "Installation erfolgreich!" sehen

2 Google Cloud Console öffnen

  • 🌐 Öffne: Google Cloud Console
  • 🔐 Melde dich mit eisfavorit@gmail.com an

3 Neues Projekt erstellen

  • Klicke oben links auf "Projekt auswählen"
  • Klicke auf "NEUES PROJEKT"
  • Name: EisFavorite Mail Export
  • Klicke auf "ERSTELLEN"
ℹ️ Das Projekt kann einige Sekunden dauern, bis es erstellt ist

4 Gmail API aktivieren

  • Gehe zu "APIs & Dienste" → "Bibliothek"
  • 🔍 Suche nach "Gmail API"
  • Klicke darauf und dann auf "AKTIVIEREN"
✅ Die Gmail API ist jetzt aktiv für dein Projekt

5 OAuth-Zustimmungsbildschirm einrichten

  • Gehe zu "APIs & Dienste" → "OAuth-Zustimmungsbildschirm"
  • Wähle "Extern" und klicke auf "ERSTELLEN"
  • Fülle aus:
    • App-Name: EisFavorite Mail Export
    • Nutzer-Support-E-Mail: eisfavorit@gmail.com
    • Entwickler-Kontakt: eisfavorit@gmail.com
  • Klicke auf "SPEICHERN UND FORTFAHREN"
  • Bei "Bereiche": "BEREICHE HINZUFÜGEN"
  • Suche "Gmail API" und wähle:
    .../auth/gmail.readonly
  • "SPEICHERN UND FORTFAHREN"
  • Bei "Testnutzer": "NUTZER HINZUFÜGEN"
  • Füge hinzu: eisfavorit@gmail.com
  • "SPEICHERN UND FORTFAHREN"
⚠️ Wichtig: Vergiss nicht, dich selbst als Testnutzer hinzuzufügen!

6 OAuth-Client-ID erstellen

  • Gehe zu "APIs & Dienste" → "Anmeldedaten"
  • Klicke auf "+ ANMELDEDATEN ERSTELLEN"
  • Wähle "OAuth-Client-ID"
  • Anwendungstyp: "Desktopanwendung"
  • Name: EisFavorite Desktop Client
  • Klicke auf "ERSTELLEN"

7 Credentials herunterladen

  • 📥 Klicke auf das Download-Symbol (↓) neben der Client-ID
  • 💾 Speichere die Datei als credentials.json hier:
    c:\Users\aagne\OneDrive\Desktop\mail.eisfavorite\credentials.json
✅ credentials.json sollte jetzt im Projektordner sein

8 Tool starten!

  • 🚀 Doppelklick auf START.bat
  • 🌐 Ein Browser-Fenster öffnet sich automatisch
  • 🔐 Melde dich mit eisfavorit@gmail.com an
  • ✅ Klicke auf "Zulassen"
  • 🎯 Wähle deine Such-Option (z.B. "1" für letzte 30 Tage)
🎉 Fertig! Das Tool exportiert jetzt deine E-Mails als JSON

9 JSON importieren

  • 📄 Eine Datei wie buchungen-export-2026-07-15.json wurde erstellt
  • 🌐 Öffne: Buchungsverwaltung
  • 📧 Klicke auf "E-Mails prüfen"
  • 📁 Wähle die JSON-Datei aus
  • ✨ Alle Buchungen werden mit "📧 Neu eingetroffen!" Badge angezeigt!
💡 Tipp: Nach dem ersten Setup musst du nur noch START.bat ausführen - die Authentifizierung wird gespeichert!
